Basics:

Basically you can use these things call p tags ("< p >text goes here< / p >") to create paragraphs like this...

Larger text use h tags ("< h >text goes here< / h >")

Formatting:

You can also format text

you can underline, bold, italicize or do all three.

Besides the main three, you can also do markup, strong text, emphasized text, small text, strikethrough text, inserted text, subscript text, and superscript text
